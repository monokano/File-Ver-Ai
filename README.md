# File Ver Ai

- macOS用デスクトップアプリ
- Adobe Illustratorファイルの作成バージョンを表示
- Illustrator編集機能を保持したPDFファイルに対応

![Aver screenshot](img/window.png)

## 使い方

ウインドウにIllustratorファイルをドロップします。Dockのアプリアイコンにもドロップできます。

## 動作環境

macOS 13.5 Ventura 以降（Universal Binary）
